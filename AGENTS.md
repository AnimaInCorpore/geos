# Agent Instructions

## Required progress tracking

- `PROGRESS.md` is the canonical tracker for the Atari XL porting status.
- When you complete any item from `PORTING.md` section "8. Recommended Porting Sequence", update `PROGRESS.md` in the same change.
- Mark completed items by changing `[ ]` to `[x]` for the exact matching step number.
- Do not reorder, renumber, or rewrite unchecked steps unless explicitly requested.
- If work spans multiple steps, check all fully completed steps and leave partial steps unchecked.
- In your final response, state which step number(s) were checked.
